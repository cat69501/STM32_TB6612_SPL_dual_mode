【软件说明】
 * 软件名称：  TB6612编码器电机驱动例程
 * MDK版本：   V5.23 
 * 固件库版本： V3.5.0
 * 主要功能：    

【硬件说明】
 * 硬件平台：X-CTR000 控制器
 * 处理器：STM32F103C8T6
 * 外部高速晶振：8MHz
 * 总线运行时钟：系统时钟 = SYCCLK = AHB1 = 72MHz
 * 硬件原理接口：请参考开源硬件原理图

【硬件连接】
 * TB6612电机驱动
   PWMA - PA8
   AIN1 - PB12
   AIN2 - PB13
   PWMB - PA11
   BIN1 - PB14
   BIN2 - PB15
   STBY - 5V（默认已连接到5V）
 
 * 电机编码器（精简模块）
   E1A - PA6  TIM3
   E1B - PA7  TIM3
   E2A - PA0  TIM2
   E2B - PA1  TIM2
   
 * ADC电压采样（1/11分压）
   ADC - PA2
